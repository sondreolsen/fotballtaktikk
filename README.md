# Fotballtaktikk

Interaktiv taktikk-tavle for 7-er fotball laget som en enkel statisk nettside.

## Funksjoner

- Velg formasjon for hvert lag: `2-3-1` eller `3-2-1`
- Flytt spillere og ball fritt rundt pa banen
- Tegn streker med piler for bevegelse eller pasningsretning
- Nullstill tavla til valgt formasjon
- Automatisk lagring i nettleseren av brikker, formasjoner og streker

## Filer

- `index.html` inneholder selve siden
- `styles.css` inneholder designet for taktikkbrettet
- `script.js` inneholder dra-og-slipp, formasjoner, tegning og lagring

## Bruk

Apne [index.html](./index.html) lokalt i nettleseren, eller bruk siden via GitHub Pages.
